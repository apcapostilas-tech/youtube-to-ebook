# EbookExpress Design System

## Brand
- **Product**: EbookExpress — transforms YouTube videos into ebooks, sales pages and Meta Ads creatives
- **Positioning**: Premium micro-SaaS, $97 product feel
- **Tone**: Bold, confident, modern, conversion-focused

## Color Palette
- Background: `#0a0a0f` (near black)
- Surface: `rgba(255,255,255,0.03)` to `rgba(255,255,255,0.08)`
- Border: `rgba(255,255,255,0.06)` to `rgba(255,255,255,0.12)`
- Primary gradient: `from-red-500 to-orange-500` (#EF4444 → #F97316)
- Text primary: `#ffffff`
- Text secondary: `rgba(255,255,255,0.6)`
- Text muted: `rgba(255,255,255,0.3)`
- Success: `#10B981`
- Error: `#EF4444`

## Typography
- Font: System sans-serif (Inter/Geist)
- Hero: `text-5xl font-black` (80px, 900 weight)
- Section title: `text-3xl font-bold`
- Card title: `text-base font-semibold`
- Body: `text-sm` with `leading-relaxed`
- Label: `text-xs uppercase tracking-wider`

## Components
- **Cards**: `bg-white/3 border border-white/5 rounded-xl p-5`
- **Buttons primary**: `bg-gradient-to-r from-red-500 to-orange-500 rounded-xl font-bold`
- **Inputs**: `bg-white/5 border border-white/10 rounded-xl px-4 py-3`
- **Badges**: `rounded-full border text-xs font-medium px-3 py-1`
- **Tabs**: `bg-white/5 rounded-xl p-1` with active `bg-white/10`

## Layout
- Max content width: `max-w-4xl mx-auto`
- Section padding: `px-6 py-10`
- Grid gaps: `gap-4` to `gap-6`
- Border radius: `rounded-xl` (12px) default, `rounded-2xl` for larger cards

## Aesthetic References
- Linear.app dark UI
- Vercel dashboard
- Framer landing page
- High-contrast, minimal, purposeful use of color
